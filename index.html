Why PikoClaw?

Built for developers who demand performance and simplicity

Blazing Fast

Written in Rust for maximum performance. No bloat, no overhead. Pure speed at the speed of thought.

Ultra Lightweight

Only 6-7 MB binary size. Download once, use anywhere. No heavy dependencies or runtime requirements.

Beautiful TUI

Gorgeous terminal interface with multiple themes. Dark, light, accessible, and ANSI variants.

Powerful Tools

11+ built-in tools including Bash, file operations, web search, Jupyter notebooks, and more.

Permission Control

Fine-grained permission system. Control exactly what the AI can do with configurable policies.

Session Management

Resume conversations seamlessly. Continue where you left off with full context preservation.

Cross-Platform

Works on macOS, Linux, and Windows. One codebase, consistent experience everywhere.

Claude Powered

Powered by Anthropic's Claude AI with support for Sonnet, Opus, and Haiku models. Switch on the fly.

MCP Support

Built-in MCP client for extensibility. Connect to external tools and services effortlessly.

Get Started in Seconds

Choose your platform and install

macOS & Linux — one-liner 
curl -fsSL https://pikoclaw.com/install.sh | bash
macOS 
# Install via Homebrew
brew tap PikoClaw/pikoclaw
brew install pikoclaw

# Set your API key
export ANTHROPIC_API_KEY=sk-ant-...

# Start coding!
pikoclaw
Linux 
# Download and install
curl -L https://github.com/PikoClaw/PikoClaw/releases/latest/download/pikoclaw-linux-x86_64 -o pikoclaw
chmod +x pikoclaw
sudo mv pikoclaw /usr/local/bin/

# Set your API key
export ANTHROPIC_API_KEY=sk-ant-...

# Start coding!
pikoclaw
Windows 
# Download from GitHub Releases
# https://github.com/PikoClaw/PikoClaw/releases

# Move to your bin directory
Move-Item pikoclaw-windows-x86_64.exe "$env:USERPROFILE\bin\pikoclaw.exe"

# Set your API key
$env:ANTHROPIC_API_KEY="sk-ant-..."

# Start coding!
pikoclaw
